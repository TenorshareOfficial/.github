## ReiBoot for iOS — what it does

ReiBoot for iOS is a desktop repair tool for iPhone and iPad. It helps with the
most common situations where a device gets stuck and normal restart steps don't
work:

- iPhone stuck on the Apple logo or in a boot loop
- Device frozen on a black or white screen
- A device trapped in Recovery Mode that won't exit
- iOS update or restore that won't finish
- Entering or exiting Recovery Mode in one click

It walks you through detection and the repair steps without manually handling
advanced restore commands.

## Quick troubleshooting reference

These are general first steps for common iOS stuck-state problems. They apply
whether or not you use a repair tool.

### iPhone stuck on the Apple logo

1. Force restart first:
   - iPhone 8 and later: press Volume Up, then Volume Down, then hold the Side
     button until the Apple logo reappears.
   - iPhone 7 / 7 Plus: hold Volume Down + Side button together.
   - iPhone 6s and earlier: hold Home + Side button together.
2. If it still loops, connect it to a computer and run a system repair so the
   correct firmware is reinstalled.

### Device stuck in Recovery Mode

1. Make sure the cable and USB port are working; try a different known-good cable.
2. If a normal force restart doesn't exit Recovery Mode, use a repair tool's
   dedicated "Exit Recovery Mode" option instead of repeatedly reconnecting.

### iOS update or restore won't finish

1. Check that the computer has a stable internet connection and enough free
   storage for the firmware package.
2. Confirm the device model and the matching iOS firmware before retrying.
3. If the update keeps failing, a guided system repair can reinstall the
   firmware cleanly.

### Device not detected

- Try another USB port and a reliable original cable.
- Unlock the device and tap "Trust" if prompted.
- Restart the repair software and reconnect.
